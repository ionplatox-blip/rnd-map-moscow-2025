## Update Instructions
To update the live site:
1. Commit your changes:
   ```bash
   git add .
   git commit -m "Update description"
   ```
2. Push to GitHub:
   ```bash
   git push
   ```
3. Render will automatically redeploy (if Auto-Deploy is on) or you can manually trigger it in the Render Dashboard.
